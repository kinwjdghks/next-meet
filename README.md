# When2meet의 재단장, next-meet

## Next-meet은 기존 when2meet(https://www.when2meet.com) 서비스의 UI/UX를 개선하고 기능을 추가한 미팅시간 선정 웹서비스입니다.

비회원으로 미팅을 만들 수 있고, 회원가입 후 미팅을 관리할 수도 있습니다.<br/>
미팅 링크를 전달받은 멤버들과 가능한 시간대를 조율하고, 일정을 확정하세요!<br/>
